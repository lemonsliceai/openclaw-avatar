# Code of Conduct

OpenClaw Avatar is committed to a respectful, inclusive, and constructive community for everyone who participates in the project.

## Expected Behavior

Community members are expected to:

- communicate respectfully and assume good intent
- give and receive feedback constructively
- be considerate of different backgrounds, skill levels, and perspectives
- respect personal boundaries, privacy, and consent
- focus discussions on improving the project

## Unacceptable Behavior

The following behavior is not acceptable in this project or its community spaces:

- harassment, intimidation, threats, or discrimination
- hateful, abusive, or demeaning language
- personal attacks, trolling, or sustained hostility
- unwelcome sexual language or attention
- doxxing or sharing someone else's private information without permission
- deliberate disruption of discussions, reviews, or project work

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and any other official communication channels for OpenClaw Avatar. It also applies when someone is representing the project in public spaces.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it to `scott@lemonslice.com`.

Please include any details that would help with review. Reports will be handled promptly and with as much confidentiality as practical.

## Enforcement

Project maintainers may take any action they believe is appropriate in response to unacceptable behavior, including:

- requesting a change in behavior
- removing content
- temporarily limiting participation
- permanently banning someone from project spaces

## Maintainer Responsibility

Project maintainers are responsible for clarifying and enforcing this Code of Conduct in good faith and in the best interests of the community and the project.
