Session

Use a session key to connect to an OpenClaw agent session.

How to pick a key: leave this blank or enter main to use the default OpenClaw session (session.mainKey, which falls back to main).

  • Enter a plain key like research if that is the session name you started in OpenClaw.
  • For the default OpenClaw main agent, the fully qualified form is agent:main:<sessionKey>, for example agent:main:main.
  • If OpenClaw already shows a full agent session key, paste it exactly as-is.
