i How to pick an image URL

Use a public image link: this should be a direct URL to an image file that is publicly accessible on the internet.

If the image is private, local to your computer, or requires a login to view, the avatar may not load correctly.

Example: https://e9riw81orx.ufs.sh/f/z2nBEp3YISrtNkoagYf5CBjh3ZkFEumULAJYeQriWT8tg79y
