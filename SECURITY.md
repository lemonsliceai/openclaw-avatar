# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in OpenClaw Avatar, please report it privately rather than opening a public issue.

**Email:** scott@lemonslice.com

Include as much detail as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a resolution timeline within 5 business days.

## Supported Versions

Security fixes are applied to the latest release only.

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
| < latest | No       |

## Scope

This policy covers the `@lemonsliceai/openclaw-avatar` npm package and the code in this repository. For vulnerabilities in OpenClaw itself, LiveKit, or LemonSlice, please report to those projects directly.
